## 📊 תוכן הדשבורד

הדשבורד כולל 3 טאבים:

1. **פרטנר Q1 2026** — KPIs, מגמות, ביצועים תפעוליים, איפה הצלחנו, איפה הבעיות

3. **חטיבה עסקית** — ניתוח B2B מעמיק כולל:

   - זכיות מכרזים — השוואה ראש-בראש
